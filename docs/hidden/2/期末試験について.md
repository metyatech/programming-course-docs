# 期末試験について

## 📋 概要

Webサイトを制作して提出して下さい。

:::danger[重要]
**この提出物のみで成績評価を行います。**
:::

:::tip[中間提出について]
紛失等に備えて、念のため、各授業の終わりに毎回、完成してなくても提出してもらいます。
:::

### デザインについて

- Webサイトのデザインについては、**自分で作成**をお願いします
- デザインは評価対象外ですが、デザイン案のファイルなども提出して下さい(進捗確認のため)
- デザインは授業の時間外で少しずつ制作を進め、**授業中はコーディングに集中**できるようにして下さい
  - 授業中、ちょこっとデザインの手直しをしたくなって少しするくらいならOKです
  - ただし、制作初回の授業のみ(デザインが無いと何もできないので)デザイン制作OKとします

:::danger[不正行為]
**他者のデザインをコピーするのは不正行為となります。**
:::

---

## 📦 最終提出物

1. **Webサイトに必要な全てのファイル**
   - HTML, CSS, JS, 画像など
2. **デザイン案のファイル**

## ❌ 禁止事項

:::danger[これらは不正行為です]

- **デザインは、他の人のデザインを単にコピーしただけのものはNG**
  - ✅ 許可を得てコピーしたうえでオリジナルに改変するのであれば良い

- **授業で渡した資料、Webの検索結果、友人、生成AI等、他人のプログラムを単にコピーして貼り付けただけのものはNG**
  - ✅ コピーしたうえで自分の制作しようとしているWebサイトにあった形に改変するのであれば良い

- **完全に他人任せでデザインやプログラムを書いてもらうのはNG**

:::

- ✅ 先生、友達、親、大人に相談するのもOK
